Symbol Enum
===

# Background

Defines constants that specify a glyph from the Segoe Fluent Icon font to use as the content of a `SymbolIcon`.

_API Notes: Glyphs with prefixes ranging from E0- to E5- are marked as legacy. Thus we are updating the enum to point to glyphs with unicode points prefixed from E7- to E9-, and **updating the font set to Segoe Fluent Icons**._
